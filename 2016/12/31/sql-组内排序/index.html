sql-利用子查询实现组内排序

发表于 2016-12-31

问题描述

table:

id name num
1 a 1
2 a 2
3 b 3
4 b 4

获得相同name的num最大的记录

错误解法

排序语法为Order By，分组语法为Group By，可能会想到：

select * from table group by name order by num

分析

group by 优先级大于 order by ，所以无法达到想要的效果

正确解法

select from (
select from table order by num desc
) group by name
